# 📝  Form – Local Storage Project

This project is a responsive registration form built using HTML, CSS, and JavaScript.  
It stores user data in the browser using **Local Storage**, allowing information to persist even after page reload.

---

## 🚀 Features

- Clean and modern UI design  
- Form validation using HTML `required` attributes  
- Gender selection using radio buttons  
- Data stored in browser Local Storage  
- Displays saved registration details dynamically  
- Option to clear saved data  
- Fully responsive layout  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
  - Flexbox  
  - Gradient backgrounds  
  - Focus effects  
  - Media Queries  
- JavaScript (Vanilla)  
  - DOM Manipulation  
  - Event Handling  
  - Local Storage API

## ▶️ How to Run

1. Download or clone the repository  
2. Open `index.html` in your browser  
3. Fill out the form and click **Register**  
4. Refresh the page to see stored data persist  
5. Click **Clear Saved Data** to remove stored information

## 🎯 Learning Focus

This project was created to practice:

- Form handling with JavaScript

- Using Local Storage to save data  
- Dynamically updating UI content  
- Creating responsive form layouts  

---

## 💡 Concept

The form demonstrates how frontend applications can store and retrieve user data locally without a backend, simulating a basic client-side registration system.
