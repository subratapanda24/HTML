# ☕ Coffee Shop – Landing Page

Coffee Shop is a simple and clean landing page built using HTML and CSS.  
The project focuses on layout structuring, Flexbox alignment, and basic responsive design concepts.

---

## 🚀 Features

- Navigation bar with logo and menu links  
- Full-width hero banner image  
- About Us section  
- Specialties section with styled list  
- Clean footer design  
- Flexbox layout for content alignment  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
  - Flexbox  
  - Box Model  
  - Hover Effects  
  - Layout Styling  

---

## 📂 Project Structure

```
Coffee-Shop/

```

---

## ▶️ How to Run

1. Download or clone the repository  
2. Open `index.html` in your browser  
3. Make sure image files are in the same folder  

---

## 🎯 Learning Focus

This project was created to practice:

- Navigation bar layout  
- Flexbox for horizontal alignment  
- Hero image styling with `object-fit`  
- Section structuring  
- Basic UI styling techniques
