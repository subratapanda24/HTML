# 📸 Golden Frame – Photography Portfolio Website

Golden Frame is a premium photography portfolio website built using HTML and CSS.  
It features a dark cinematic theme with gold accents and a modern, luxury-inspired layout.

---

## ✨ Features

- Fixed transparent navigation bar  
- Full-screen hero section with overlay effect  
- About Me section with portrait image styling  
- Responsive grid-based portfolio gallery  
- Image hover zoom + overlay effect  
- Contact form layout  
- Elegant typography using Google Fonts  

---

## 🎨 Design Theme

- Dark luxury background  
- Gold accent color palette  
- Script + Serif typography combination  
- Cinematic photography aesthetic  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
  - Flexbox  
  - CSS Grid  
  - Hover Effects  
  - Media Queries  
  - CSS Variables  

---

## ▶️ How to Run

1. Clone or download the project  
2. Open `index.html` in your browser  
3. Scroll to explore sections and gallery
