# 🌐 Personal Portfolio Website

This is a simple personal portfolio website built using HTML and CSS.  
It showcases personal information, skills, projects, and contact links in a clean and minimal layout.

---

## 🚀 Features

- Profile card with circular image  
- Clickable profile image linked to LinkedIn  
- Audio introduction section  
- Contact links (Email, Phone, LinkedIn, GitHub)  
- About Me section  
- Skills list  
- Projects section  
- Clean and minimal design  
- Responsive centered layout  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- Embedded Audio  
- External Links (LinkedIn, GitHub, Email, Phone)

## ▶️ How to Run

1. Download or clone the repository  
2. Open `index.html` in your browser  
3. Click the profile photo to visit LinkedIn  
4. Play the audio introduction  
5. Use contact buttons to connect
