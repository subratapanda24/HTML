# ✨ Motion – Creative Landing Page with Canvas Animation

Motion is a modern SaaS-style landing page built using HTML, CSS, and JavaScript.  
It combines clean UI design with a custom Canvas-based animation to create an interactive and visually engaging hero section.

---

## 🚀 Features

- Fixed navigation bar with backdrop blur  
- Responsive hero layout (text + animated canvas)  
- Custom Canvas illustrations (Notion-style windows & users)  
- Animated bouncing characters  
- Dynamic dashed sync lines  
- Responsive typography using Inter font  
- Clean, SaaS-inspired UI design  
- Fully responsive across desktop, tablet, and mobile  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
  - Flexbox  
  - Media Queries  
  - CSS Variables  
  - Backdrop Filter  
- JavaScript (Vanilla)  
- HTML5 Canvas API  
- Google Fonts (Inter)

## ▶️ How to Run

1. Download or clone the repository  
2. Open `index.html` in your browser  
3. Resize the window to see responsive behavior  
4. Watch the animated canvas illustration in the hero section
