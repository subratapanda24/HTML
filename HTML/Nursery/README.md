# 🌿 Petal & Root – Botanical Nursery Website

Petal & Root is a modern botanical nursery website built using HTML, CSS, and JavaScript.  
It features an elegant plant catalog layout with soft botanical colors, interactive filtering, and a clean, premium aesthetic.

---

## ✨ Features

- Sticky header with clean navigation  
- Elegant hero section with botanical typography  
- Category filter system (All, Perennial, Annual, Tropical, Wildflower)  
- Responsive CSS Grid plant gallery  
- Interactive save (favorite) button  
- Add-to-cart styled UI buttons  
- Newsletter subscription section  
- Smooth hover effects and transitions  

---

## 🎨 Design Theme

- Cream and parchment background tones  
- Moss green and terracotta accent colors  
- Serif + Sans-serif typography pairing  
- Soft shadows and rounded card layouts  
- Minimal, modern botanical aesthetic  

---
