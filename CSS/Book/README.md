# 📚 BookNest – Premium Bookstore Landing Page

BookNest is a modern bookstore landing page built using HTML and CSS.  
It showcases featured books, cozy reading spaces, and a clean premium UI layout.

---

## 🚀 Features

- Elegant navigation bar with search section  
- Hero section with background overlay  
- Featured books displayed in a responsive flex layout  
- Cozy reading corner gallery using CSS Grid  
- Clean typography and warm color palette  
- Responsive design for smaller screens  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3 (Flexbox + Grid + Media Queries)
