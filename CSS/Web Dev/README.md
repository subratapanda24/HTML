# 🛍️ ESTATE – Curated Goods E-Commerce Landing Page

ESTATE is a premium, minimal e-commerce landing page designed to showcase curated lifestyle products.  
The project focuses on modern layout systems, typography pairing, and refined UI aesthetics using pure HTML and CSS.

---

## 🚀 Features

- Sticky navigation with backdrop blur effect  
- Elegant hero section with split layout  
- Product grid with hover zoom interaction  
- Trust bar highlighting brand values  
- Responsive grid system using CSS Grid  
- Newsletter subscription section  
- Premium typography using Google Fonts  
- Smooth hover transitions and micro-interactions  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
  - CSS Variables  
  - Flexbox  
  - CSS Grid  
  - Responsive Design  
  - Transitions & Hover Effects  
- Google Fonts (Playfair Display & Montserrat)  
- Font Awesome Icons  

---

## ▶️ How to Run

1. Clone or download the repository  
2. Open `index.html` in your browser  
3. Internet connection required for Google Fonts, Font Awesome, and Unsplash images
