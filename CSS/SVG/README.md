# 🎨 SVG Practice – Shapes & Animations

This project is a collection of SVG practice examples created to explore vector graphics, shapes, paths, gradients, and animations using pure HTML.

---

## 🚀 What This Project Covers

- Basic SVG shapes:
  - Circle
  - Rectangle
  - Line
  - Polygon
  - Path
  - Text

- Advanced SVG concepts:
  - Quadratic & Cubic Bézier curves
  - Stroke styling (dasharray, dashoffset)
  - Opacity animation
  - Transform animations (scale)
  - Color animations
  - Linear & Radial Gradients
  - Animated gradient effects
  - Drawing animation effect (stroke reveal)

---

## 🛠️ Technologies Used

- HTML5  
- SVG (Scalable Vector Graphics)  
- SVG Animation (`<animate>`, `<animateTransform>`)  

---

## 📂 Project Structure

```
SVG-Practice/
│
└── index.html
```

(All SVG examples are written directly inside a single HTML file.)

---

## ▶️ How to Run

1. Open the HTML file in your browser  
2. No installation required  
3. Works in any modern browser  

---

## 🎯 Purpose

This project was created to:

- Understand how SVG works inside HTML  
- Practice creating shapes using coordinates  
- Learn SVG animation techniques  
- Experiment with gradients and visual effects
