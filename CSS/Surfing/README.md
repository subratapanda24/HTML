# 🌊 WaveRiders – Surf Academy Landing Page

WaveRiders is a modern and responsive surf academy landing page built using HTML5 and CSS3.  
The project focuses on clean UI design, animations, and responsive layout techniques.

---

## 🚀 Features

- Sticky navigation bar with hover underline effects  
- Full-screen hero section with background overlay  
- Animated logo using CSS keyframes  
- Course cards with hover transformations  
- Image gallery with grayscale hover effect  
- Section numbering using CSS counters  
- Responsive typography using `clamp()`  
- Smooth scrolling behavior  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
  - Flexbox  
  - CSS Grid concepts  
  - Media techniques  
  - CSS Variables  
  - Animations & Transitions  
  - Pseudo-elements  
  - CSS Counters
