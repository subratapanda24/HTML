# 🚀 2026 — Forward System (Canvas Animation)

**2026 — Forward System** is an interactive futuristic canvas animation built using pure HTML5 Canvas and JavaScript.  
It visualizes motion, direction, and time through animated particles, planetary systems, and a dynamic timeline.

---

## 🌌 Features

- Animated starfield background  
- Custom-drawn Saturn and Jupiter using Canvas API  
- Interactive timeline wave influenced by mouse movement  
- Particle motion system with layered depth effect  
- Dynamic text animation with glow and breathing effect  
- Grid overlay and scanline visual styling  
- Smooth animation loop using `requestAnimationFrame`  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla)  
- HTML5 Canvas API  

---

## 📂 Project Structure

```
Forward-System/
│
└── index.html   (Contains HTML, CSS, and JavaScript in one file)
```

---

## ▶️ How to Run

1. Download or clone the repository  
2. Open `index.html` in any modern browser  
3. Move your mouse across the canvas to interact with the timeline
