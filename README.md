# HTML and CSS Project Collection — Subrata Panda

This repository contains multiple frontend practice projects built using HTML5 and CSS3.  
Each folder represents an independent website layout or design experiment.

The projects focus on structured layouts, responsive design, and modern UI styling using pure HTML and CSS.

---

## 📂 Project Structure

Projects are organized into two main categories:

```
CSS/
│
├── BookShop/
├── Canvas2026/
├── SVGs/
├── Surfing/
└── WebDev/

HTML/
│
├── CoffeeShop/
├── DynamicSite/
├── Form/
├── Nursery/
├── Photography/
└── Portfolio/
```

Inside each category, every project folder contains its own HTML files, CSS files, and related media assets such as images, audio, or video files where required.

File names may vary depending on the project.

---

## 🚀 Key Highlights

- Multi-section website layouts  
- Responsive design using Flexbox and CSS Grid  
- Hero sections and navigation systems  
- Card-based layouts and galleries  
- Media integration (images, audio, background video)  
- Clean typography using Google Fonts  
- Reusable styling with CSS variables

## 🛠️ Technologies Used

- HTML5  
- CSS3  
  - Flexbox  
  - CSS Grid  
  - Media Queries  
  - CSS Variables  
  - Transitions and Hover Effects

## ▶️ How to Run

1. Clone or download the repository  
2. Open any project folder  
3. Open the main HTML file of that project in your browser  

No installations or external dependencies are required.
